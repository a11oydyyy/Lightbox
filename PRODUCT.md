# Lightbox

Lightbox is a native macOS image material browser for local folders and NAS-backed product catalogs. The Full branch keeps the clean Lite gallery experience, then adds Finder-like source browsing, pinned folders, folder navigation, and a local SQLite index.

The product prioritizes native macOS feel, refined Light Mode surfaces, quiet controls, fast folder traversal, and image-first comparison workflows. The local Library stores imported assets for offline use. Pinned folders are referenced in place. Tags follow macOS Finder color tags only.

This branch intentionally does not add accounts, cloud sync, plugins, video management, AI tags, or permanent file deletion.
