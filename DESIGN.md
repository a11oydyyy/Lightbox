# Lightbox Design System

The interface is a refined macOS product UI optimized for Light Mode. Glass is used for navigation and floating controls only: the top path bar, source selector, bottom scale control, scroll indicator, preview information, and drop/import status. The gallery background stays white, calm, and cool-neutral so images remain the primary content.

Motion is short, polished, and state-driven: path bar morphing between navigation and selection state, bottom control hover/drag, directory row reveal, scroll indicator fade, subtle card hover tilt, preview open/close scale, and preview left/right transitions. Reduced motion falls back to short fades.

Typography uses English system text, compact sizes, and clear weight changes. Gallery cards do not show file names or tags. Directory tiles are compact and do not scale with image thumbnails. Preview shows only file name, resolution, and Finder color tags. The UI avoids marketing copy, oversized display type, decorative cards, and heavy custom controls.
